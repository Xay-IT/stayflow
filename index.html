🔗 Connect to Google Sheets

Paste the Web App URL from Google Apps Script to sync bookings in real-time

1. Open Google Sheets → Extensions → Apps Script → Paste code below → Save
2. Click Deploy → New Deployment → Web App
3. Set: Execute as: Me  |  Who has access: Anyone ← Important!
4. Click Deploy → Copy Web App URL → paste in the field above
⚠️ After editing code, always Deploy as new version each time
